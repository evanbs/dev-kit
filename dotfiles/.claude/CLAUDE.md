# Perfil de Desenvolvedor

## Quem sou

Desenvolvedor fullstack com foco em frontend. TypeScript é minha linguagem principal.
Trabalho com Node.js, React/frameworks modernos, e eventualmente .NET, Python e Lambdas.

## Preferências gerais

- TypeScript em tudo que for possível
- Código explícito > código "mágico"
- Componentes pequenos e focados
- Testes quando fazem sentido, não por obrigação de cobertura

## Como trabalho com agentes

Prefiro o modo semi-autônomo: tarefas bem definidas, execução autônoma, revisão antes
de ações permanentes. Uso `tlc-spec-driven` para contexto e docs de projeto — se não
souber o que é, leia `.claude/skills/tlc-spec-driven/SKILL.md` ou `.kiro/skills/`.
